# TPI: Integración de Matemática y Programación I 🐍🔢

Este proyecto consiste en un programa desarrollado en **Python** que unifica los conocimientos adquiridos en las materias de **Matemática** y **Programación I** de la Tecnicatura Universitaria en Programación (UTN).

El objetivo es aplicar lógica de programación para resolver y simular fenómenos matemáticos específicos vistos en las primeras tres unidades de la cursada.

## 🚀 Funcionalidades

El programa incluye herramientas interactivas para:
1. **Sistemas de Numeración (Unidad 1):** - Conversión de números Decimales a Binarios mediante el método de divisiones sucesivas.
   - Conversión de números Binarios a Decimales mediante el método de potencias de base 2.
2. **Álgebra de Boole (Unidad 2):** - Implementación de funciones que simulan compuertas lógicas (AND, OR, NOT, XOR).
3. **Lógica Proposicional (Unidad 3):** - Verificador de tablas de verdad para conectivos lógicos (Condicional, Bicondicional, etc.).

## 🛠️ Tecnologías utilizadas

- **Lenguaje:** Python 3.x
- **Entorno:** Linux (ejecutado sobre Linux Mint)
- **Conceptos aplicados:** Funciones, tipos de datos complejos (listas), estructuras de control de flujo (while/for), y Type Hinting.
